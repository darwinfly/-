# 產品展示網站 更新日誌 (CHANGELOG)

本專案採用語意化版本號（Semantic Versioning）管理。

## [v1.5.4] - 2026-07-11
### 修正
- 後台全域設定覆寫失效修正：將全域設定取得函數 `db_get_setting` 的優先級邏輯調整為「優先讀取資料庫，其次環境變數，最後預設值」。修復了因環境變數優先級高於資料庫而導致後台修改網站名稱與主副標題點擊儲存無效的問題。

## [v1.5.3] - 2026-07-11
### 安全性與組態更新
- 後台管理密碼同步機制：重構後端資料庫初始化 `init_db()` 方法。系統每次冷啟動時，若環境變數中存在 `ADMIN_PASSWORD`，將無條件自動同步/覆寫資料庫中的管理員（admin）密碼為該值，以保障兩者的一致性。
- 預設登入密碼修改：在 `.env`、`.env.example` 與 GCP Cloud Run 服務中，將預設管理者登入密碼變更為 `admin1234`。

## [v1.5.2] - 2026-07-11
### 新增
- 品牌與標題全動態化：
  - 將頂部 Logo 品牌名稱（`SITE_NAME`）納入全域配置，與首頁主標題（`SITE_TITLE`）和副標題（`SITE_SUBTITLE`）共同組成全站設定系統。
  - 後端以 Flask `context_processor` 實作全域變數注入，所有 HTML 模板（base, index, detail, admin）皆無縫讀取，簡化代碼結構。
  - 後台「網站資訊設定」面板擴充「網站名稱」欄位，支援線上即改即用。
  - **客製化彩蛋設定**：預設品牌設定為 `Antigravity Showcase`，搭配物理突破科技主副標題。

## [v1.5.1] - 2026-07-11
### 新增
- 網站首頁標題設定面板：於 `/admin` 後台新增「網站首頁標題與副標題設定」面板。管理者可直接在後台輸入框更新主標題與副標題，無需重新編譯與部署。
- 全域設定資料庫支援：後端新增 `settings` 抽象儲存層（Firestore / SQLite）與 `db_get_setting`、`db_set_setting` 輔助方法。優先讀取環境變數配置，其次自資料庫加載設定。
- 預設文字配置：將原主標題與副標題預設值改為使用者指定之 `[請在此輸入你想換的新主標題文字]` 與 `[請在此輸入你想換的新副標題文字]`。

## [v1.5.0] - 2026-07-11
### 新增
- Google 雲端硬碟 (Google Drive) 深度整合：
  - 雲端選擇圖片：在後台「產品展示圖片」選取區，新增「從雲端選擇圖片」按鈕。支援從 Google Drive 選擇 JPG、PNG、WebP 等圖檔。
  - 雲端選擇簡報：在後台「上傳 PDF 簡報」選取區，新增「從雲端選擇簡報」按鈕。支援選擇 PDF 文件，並**自動將 Google Slides (簡報) 在線匯出為 PDF 檔案流**匯入儲存。
  - 技術實作：前端引入 Google Picker API 與最新 Google Identity Services (GIS) 進行 OAuth 2.0 授權；後端實作下載/匯出路由與二進位 Bytes 流儲存。

## [v1.4.5] - 2026-07-11
### 修正
- PDF 預覽視窗互動修正：
  - 關閉失效修復：為頂部工具列 `.pdf-viewer-header` 設定更高的 z-index 與相對定位，使其浮在透明保護層之上，確保「關閉預覽」按鈕點擊正常。
  - 無法滾動修復：為 `.pdf-protect-overlay` 保護層設定 `pointer-events: none`，允許滑鼠滾輪滾動與點擊事件穿透到 `iframe` 內部以利流暢滾動閱讀。同時，透過先前實作的 `#toolbar=0`、全網頁禁右鍵與 Ctrl+S/P 下載阻斷，繼續提供可靠的防下載保護。

## [v1.4.4] - 2026-07-11
### 優化
- 首頁卡片圖片樣式修正：將 `object-fit` 從 `cover` 調整為 `contain`，解決去背藥瓶、包裝盒等直式產品照片頂部與底部被裁切不全的問題。
- 視覺美化：圖片容器（wrapper）高度提升至 280px，加入 `padding: 16px` 內邊距，並將背景色改為純白色（`#ffffff`），使所有去背產品能自然融合，呈現精美整齊的櫥窗效果。

## [v1.4.3] - 2026-07-11
### 修正
- 圖片路徑解析與防護：將圖片的路徑解析與回退邏輯整合至後端 `_row_to_product`。不論是雲端 GCS 網址、本地相對路徑，或是新產品未設定圖片時，都彙整為權威的 `product.image` 屬性。
- 視圖乾淨化：重構首頁、詳細頁與後台的 HTML 模板，直接讀取 `product.image` 渲染，徹底解決新創產品首頁與詳細頁破圖的 Bug，並確保相對路徑永不失效。

## [v1.4.2] - 2026-07-11
### 新增
- 簡報刪除功能：後台各產品 PDF 清單與全站 PDF 總覽列表中，新增「刪除此簡報」按鈕。
- 儲存連動清除：點擊刪除後會跳出二次確認彈窗，POST 到後端路由同步清除資料庫紀錄，並自 GCS 或本地資料夾中永久刪除實體 PDF 檔案，避免殘留垃圾檔案佔用雲端空間。

## [v1.4.1] - 2026-07-11
### 修正
- 預覽修復：移除了 iframe 中的 `sandbox` 限制屬性，修復了 GCS 上傳 PDF 後部分瀏覽器內建 PDF 閱讀器載入空白/灰色的問題。
- 工具列控制：在 PDF 預覽網址結尾加上 `#toolbar=0&navpanes=0`，強制隱藏內建的下載/列印工具列，確保「只預覽不提供下載」的安全防護機制。

## [v1.4.0] - 2026-07-11
### 新增
- 權限管理優化：將 PDF 拖曳上傳功能從前台詳細頁面移除，遷移至 `/admin` 後台管理者專屬區。
- 前台檢視限制：移除了前台所有的 PDF「下載」按鈕，並加上透明防護層、禁用右鍵另存與禁用 Ctrl+S/P 下載。
- 動態產品管理 (CRUD)：後台新增「＋ 新增產品」與「刪除此產品」功能，允許管理者彈性新增/移除展示產品。
- 圖片託管雲端化：上傳的產品展示圖在雲端模式下將自動上傳至 GCS，並保存雲端網址於 Firestore，解決 Cloud Run container 重啟圖片遺失問題。

## [v1.3.0] - 2026-07-11
### 新增
- GCP 雲端整合：實作 Google Cloud Storage (GCS) 上傳支援，將 PDF 直接上傳至雲端儲存空間。
- GCP Firestore 整合：實作 Google Cloud Firestore 原生 NoSQL 資料庫支援，用於儲存管理者密碼雜湊與 PDF 檔案的雲端 URL 基本資訊。
- 雙模切換架構：透過環境變數 `DB_MODE` (sqlite / firestore) 與 `STORAGE_MODE` (local / gcs) 提供無縫本地與雲端雙模切換，保護開發測試環境。

## [v1.2.0] - 2026-07-11
### 新增
- 密碼安全防護優化：引入 SQLite 資料庫儲存帳戶，並使用 Werkzeug 加密雜湊機制 (`generate_password_hash` 和 `check_password_hash`) 取代明文比對。
- 密碼修改功能：在 `/admin` 後台新增密碼修改面板，提供舊密碼、新密碼、確認新密碼的完整表單與後端校驗。
- 資料庫自動初始化：第一次啟動時，若 `users` 表為空，會自動建立管理員帳戶並生成預設密碼（優先讀取環境變數 `ADMIN_PASSWORD`，否則使用 `admin123` 作為預設初始密碼）。

## [v1.1.0] - 2026-07-11
### 新增
- 密碼登入機制：新增 `/login` 與 `/logout` 頁面與路由，支援環境變數 `ADMIN_PASSWORD` 安全驗證。
- 管理者後台：新增 `/admin` 頁面，未登入前自動攔截重導向至登入頁。
- 產品資料持久化：引進 `products.json` 保存產品狀態，後台編輯後可立即持久化寫入。
- 後台管理功能：支援產品名稱、價格、簡介、規格參數、以及 AI 展示圖片的更新與上傳。
- GCP 部署就緒：移除寫死的祕鑰與密碼，完全依賴 `dotenv` 與 `os.environ` 機制。

## [v1.0.0] - 2026-07-11
### 新增
- 專案基礎骨架與 Flask 後端開發。
- 磨砂玻璃擬態設計（Glassmorphism）與 Vanilla CSS 現代化響應式介面。
- 產品首頁：展示耳機、投影機、手錶等產品卡片與滑鼠懸停微動畫。
- 產品內頁：展示詳細規格、價格，並包含 PDF 規格書上傳功能。
- PDF 管理功能：支援 PDF 上傳（儲存於 `static/uploads/`）、在線檢視與點擊下載。
- 版本號展示：在全站頁尾顯示目前的版本號（`v1.0.0`）。
